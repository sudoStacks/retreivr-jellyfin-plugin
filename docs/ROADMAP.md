# Roadmap

## Phase 1

- Connect to Resolution API
- Read plugin configuration from Jellyfin admin page
- Resolve single recording MBID
- Bulk resolve recording MBIDs
- Normalize availability into Jellyfin-friendly display states

## Phase 2

- Attach availability badges to track/album/artist views
- Add manual “Resolve via Retreivr” action
- Add manual “Download via Retreivr” action

## Phase 3

- Instant play path for `verified` and `local_only`
- Artist radio using bulk resolve
- Music video availability and playback

## Phase 4

- Local caching inside the plugin
- Snapshot / diff sync for lower latency
- contributor/reputation/network diagnostics view
